# A Multi-Tissue Atlas of Exercise and Temporal smORF Regulation

This repository contains code and supplementary data for the manuscript.

## Repository Structure

```
├── code/
│   ├── alignment/              # Read alignment and assembly scripts
│   ├── counting/               # Read quantification scripts
│   ├── differential_expression/ # DESeq2 and tau analysis pipeline
│   └── figure_generation/      # Figure generation scripts
├── data/
│   ├── gene_annotation_files/  # GTF annotations for community use
│   ├── sankey/                 # Classification data for Sankey diagrams
│   └── supplementary/          # Supplementary tables (CSV)
└── notebooks/           # Jupyter notebook for atlas construction
```
